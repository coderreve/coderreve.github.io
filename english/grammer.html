                            # 英语语法的大体框架

                            参考 知乎: 英语语法的框架open in new window

                            首先: 英语 = 句子+句子

                            所以研究英语语法就是研究句子的组成与结构

                            我们一般从下面三种角度研究英语句子

                            1. 一分为二法: 英语句子 = 简单句+修饰成分?

                              其中简单句是必须的, 修饰成分是可选的. 有修饰成分的英语句子又称为长难句, 意思为又长又难的英语句子...

                            2. 句子是由一个个的单词组成的

                              这种方法是在研究每个单词的性质, 将单词分为十种: 名词, 动词, 冠词...

                            3. 句子的结构

                              将句子根据某些特定特性划分为若干段, 每一段有自己独特的特点, 共有八段: 主语, 谓语, 宾语, 定语...

                            # 什么是简单句

                            简单句是英语句的必须的成分. 是主干. 就好比茂密树木, 树干是简单句, 叶子是修饰成分 就好比网页, html是简单句, css是修饰成分

                            # 定语从句

                            定语用于限定人,物. 若定语是以一个单独句子出现的, 称为定语从句

                            定语从句修饰的对象, 称为先行词.

                            • 比如笨猴子, 笨就是猴子的定语, 猴子称为先行词

                            限定性定语从句: 限制对人物的描述(缩窄人物的范围).

                            • 如一名教数学的老师. 老师是主语, 教数学的是定语, 并且缩窄了老师的范围, 是限制性定语从句

                            非限制性定语从句: 对人物描述的补充(详细说明)

                            • 不断快速发展的中国. 北京是主语, 不断快速发展的是定语, 并且对中国进行了详细说明
                            • 非限制性定语从句中的主语/宾语和后面的定语用逗号连接

                            不断快速发展的中国, 这句话中如果翻译成英文, 在中国(主语)和快速发展的(定语从句)之间会出现一个词用于连接. 这个词没有实际意思, 只在英语语法中起连接词, 并且中文中并没有.

                            根据连接词不同可以将定语从句分为下列几种:

                            that引导的定语从句

                            who/whom引导的定语从句

                            which引导的定语从句

                            why引导的定语从句

                            when引导的定语从句

                            where引导的定语从句
