        # 因特网概述

        网络, 互联网, 因特网

        网络: 节点(指设备:手机, 平板等)和链路(有限线路或无线线路)构成

        互联网: 多个网络之间通过路由器链接形成互联网

        因特网: 世界上最大的互联网

        internet 与 Internet 前者指互联网, 后者指因特网

        因特网发展的三个阶段

        1. 美国内部网APPANET
        2. NSFNET 三级结构因特网(主干, 地区, 校园网)
        3. 基于ISP多结构的因特网

        ISP internet service provider 互联网服务提供商, 因特网商用后, 用户需要向ISP申请互联网服务 中国主要的ISP有中国电信, 中国联通, 中国移动

        因特网标准化工作 最终形成RFC文档, 形成的过程为 草案 --> 建议(从这一步进入RFC文档) --> 标准

        # 电路交换, 分组交换, 报文交换

        指的是主机之间交换数据的三种方式

        1. 电路交换: 预先分配资源, 建立长链接, 通讯结束, 断开链接(如早期的拨号电话, 有人占用着线路, 别人 就没法打电话)
        2. 报文交换. 报文指数据. 报文交换是将整个报文通过交换网络发送到目的主机, 对交换网络节点的缓存 能力要求高
        3. 分组交换. 将报文分成若干个组, 并加上首部依次发送通过交换网络发送目的主机, 对交换网络节点的 缓存能力要求较低, 不一直占用资源. 发送方需要构造分组, 交换节点需要存储转发分组, 接受方 需要还原分组
